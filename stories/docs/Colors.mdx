## Usage

```tsx
// Use CSS custom properties in Tailwind
<div className="bg-[var(--color-primary)] text-[var(--color-primary-foreground)]">
  Primary background
</div>

<div className="border border-[var(--color-border)]">
  Bordered element
</div>
```
