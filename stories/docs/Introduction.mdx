# Soilmass Design System

Welcome to the **Soilmass Design System** component library. This is the reference implementation built with:

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **TypeScript** (strict mode)

## Architecture

The component library follows atomic design principles:

### Atoms
Foundational building blocks: Button, Input, Checkbox, Radio, Badge, Avatar, etc.

### Forms
Form-related components: FileUpload, NumberInput, SearchInput, TagInput, etc.

### Feedback
User feedback components: Alert, Toast, Progress, Skeleton, EmptyState, etc.

### Navigation
Navigation components: Tabs, Breadcrumb, Pagination, Stepper, etc.

### Overlays
Modal dialogs and popups: Modal, Drawer, Dropdown, Popover, Tooltip, etc.

### Data Display
Data presentation: Table, CodeBlock, Timeline, Calendar, etc.

### Layout
Layout components: Divider, BentoGrid, etc.

### Patterns
Composed patterns: Cards, Headers, Footers, etc.

### Sections
Marketing sections: Hero, Features, Pricing, Testimonials, etc.

### Effects
Visual effects: Spotlight, Glass, Particles, Animations, etc.

## Design Tokens

All components use CSS custom properties (design tokens) for consistent styling:

```css
--color-primary: #2563eb;
--color-surface: #ffffff;
--color-text: #0a0a0a;
--radius-md: 0.5rem;
--space-4: 1rem;
```

- ✅ **WCAG 2.1 AA** accessibility compliant
